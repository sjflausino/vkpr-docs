# Security

Currently, companies tend to have sensitive data that must be stored and controlled.

VKPR offers certain types of Security, being them:

- [Cert-Manager](https://cert-manager.io/docs/): Cert-Manager adds, renews, and validates certificates and certificate issuers as resource types in Kubernetes clusters. To see cert-manager commands, click [here](/docs/commands/cert-manager/install/aws)
- [External-DNS](https://github.com/kubernetes-sigs/external-dns): ExternalDNS synchronizes exposed Kubernetes Services and Ingresses with DNS providers. To see External-DNS commands, click [here](/docs/commands/external-dns/install/aws)
- [Keycloak](https://www.keycloak.org/documentation): Keycloak is an access and identity management solution. To see Keycloak commands, click [here](/docs/commands/keycloak/install)
- [Vault](https://www.vaultproject.io/docs): HashiCorp Vault is an identity-based secrets and encryption management system. A secret is anything that you want to tightly control access to, such as API encryption keys, passwords, and certificates. To see Vault commands, click [here](/docs/commands/vault/install)
- [External-secrets](https://external-secrets.io/v0.6.0-rc1/): External Secrets Operator is a Kubernetes operator that integrates external secret management systems like AWS Secrets Manager, HashiCorp Vault, Google Secrets Manager, Azure Key Vault and many more. The operator reads information from external APIs and automatically injects the values into a Kubernetes Secret.
