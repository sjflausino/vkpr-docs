# VKPR devportal demo install

## Description

Install the DevPortal's chart demo on an Okteto cluster. This demonstration will use Github (for code repositories) and Okteto Cloud (for the runtime), for the sake of being freely available for everyone. For more information about devportal, click [here](https://docs.platform.vee.codes/devportal/intro).

## Commands

Interactive inputs:

```bash
  vkpr devportal demo install [flags]
```

## Argument Reference

### Flags

```sh
  --github_client_organization    Specifies the organization or user on github
  --github_client_id              Specifies the GitHub OAuth App Client ID 
  --github_client_secret          Specifies the GitHub OAuth App Client Secret 
  --github_token                  Specifies the GitHub access token
  --password                      Specifies the password for the database that will be created for the devportal  
```
