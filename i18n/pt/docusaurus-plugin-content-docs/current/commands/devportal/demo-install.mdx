# VKPR devportal demo install


## Descrição

Instale o gráfico de demonstração do DevPortal em um cluster Okteto. Esta demonstração usará Github (para repositórios de código) e Okteto Cloud (para tempo de execução), para estar disponível gratuitamente para todos. Para obter mais informações sobre o devportal, clique [aqui](https://docs.platform.vee.codes/devportal/intro).

## Comandos

Comando interativo:

```bash
  instalação de demonstração do vkpr devportal [sinalizadores]
```

## Referência de valores

### Flags

```sh
  --github_client_organization      Especifica a organização ou usuário no github
  --github_client_id                Especifica o GitHub OAuth App Client ID 
  --github_client_secret            Especifica o GitHub OAuth App Client Secret 
  --github_token                    Especifica o token de acesso do GitHub
  --password                        Especifica a senha do banco de dados que será criado para o devportal
```
