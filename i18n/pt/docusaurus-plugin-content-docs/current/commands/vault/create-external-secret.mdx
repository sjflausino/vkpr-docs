# VKPR vault create external-secret

## Descrição

Este comando criará uma ExternalSecret que sincronizará com as secrets do Vault por meio do SecretStore.

## Comandos

```bash
$ vkpr vault create external-secret

Formula was successfully built!
? ExternalSecret name: vkpr-vault
? ExternalSecret namespace: vkpr
? SecretStore reference: vkpr-vault-backend
? Secret store namespace: (leave empty for ClusterSecretStore reference) [? for ? Secret store namespace: (leave empty for ClusterSecretStore reference) 
? Secret path: kv/path
? Secret key: key
...
```

Após a execução, será criado uma secret e uma ExternalSecret que será responsável por sincronizar a secret criada com uma secret remoto no provedor selecionado anteriormente.
